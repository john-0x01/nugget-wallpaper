Nugget + iPhone / iPad

Apple 기기에 라이브 배경화면 적용
탈옥은 필요 없습니다

Nugget을 사용하면 iPhone과 iPad에 애니메이션 배경화면을 간단한 절차로 가져올 수 있습니다. 외부 월페이퍼 라이브러리에서 원하는 리소스를 선택해 빠르게 적용하세요.

월페이퍼 라이브러리 열기 Nugget GitHub 보기

이 방식의 장점

  • 탈옥 없이 사용할 수 있어 일상 사용에 적합합니다.
  • iPhone과 iPad 모두를 고려한 워크플로우입니다.
  • 고품질 라이브 배경화면 리소스를 쉽게 확보할 수 있습니다.
  • 오픈소스 프로젝트라 업데이트 추적이 쉽습니다.

핵심 기능

복잡함을 줄이고 결과에 집중한 현대적이고 미니멀한 흐름입니다.

탈옥 불필요

시스템 안정성을 유지하면서 표준 절차로 배경화면을 적용합니다.

iPhone + iPad 지원

iOS와 iPadOS에서 일관된 사용자 경험을 제공합니다.

빠른 가져오기

다운로드 후 Nugget으로 쉽게 가져와 즉시 적용할 수 있습니다.

월페이퍼 라이브러리

공식 외부 라이브러리에서 최신 콘텐츠를 탐색하세요.

사용 방법

선택부터 적용까지 일반적으로 몇 분이면 완료됩니다.

1

배경화면 선택

wallpaper.nugget.host 에서 원하는 라이브 배경화면을 다운로드합니다.

2

Nugget으로 가져오기

Nugget 안내에 따라 Apple 기기에 리소스를 가져옵니다.

3

기기에서 활성화

iOS/iPadOS 배경화면 설정에서 적용하고 모션 효과를 확인합니다.

설치 가이드와 릴리스 노트를 확인하려면: Nugget GitHub 저장소 열기

자주 묻는 질문

탈옥이 꼭 필요한가요?

아니요. 이 페이지는 Nugget 기반의 비탈옥 워크플로우를 설명합니다.

어떤 기기를 지원하나요?

주요 대상은 iPhone과 iPad입니다. 상세 호환성은 Nugget 저장소를 확인하세요.

월페이퍼 사이트와 Nugget은 같은 서비스인가요?

아닙니다. 월페이퍼 라이브러리와 Nugget 프로젝트는 별도이며, 이 페이지에서 함께 안내합니다.
