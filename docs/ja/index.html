Nugget + iPhone / iPad

Apple デバイスをライブ壁紙で演出
脱獄は不要

Nugget を使えば、iPhone・iPad にアニメーション壁紙を簡単に導入できます。外部壁紙ライブラリから素材を選び、短時間で適用できます。

壁紙ライブラリを開く Nugget GitHub を見る

この構成のメリット

  • 脱獄なしで運用でき、日常利用に向いています。
  • iPhone と iPad の両方に対応した導入フロー。
  • 外部ライブラリから高品質な素材を入手可能。
  • オープンソースなので更新状況を追いやすい。

主な機能

モダンでミニマルな導入体験。不要な手順を減らし、結果までを最短化します。

脱獄不要

システム改変なしで、標準的な手順でライブ壁紙を導入できます。

iPhone / iPad 対応

iOS と iPadOS で一貫した操作感を維持します。

高速インポート

ダウンロード後に Nugget で素早く取り込み可能です。

壁紙ライブラリ

公式外部ライブラリで新しい壁紙を継続的に取得できます。

使い方

選択から適用まで、通常は数分で完了します。

1

壁紙を選ぶ

wallpaper.nugget.host でライブ壁紙をダウンロードします。

2

Nugget で導入

Nugget の手順に従って Apple デバイスへ素材をインポートします。

3

端末で有効化

iOS / iPadOS の壁紙設定で適用し、動きのある表示を楽しめます。

セットアップ詳細や更新履歴を確認する場合: Nugget GitHub リポジトリへ

不要です。このページは Nugget を使った非脱獄フローを前提としています。

主に iPhone と iPad 向けです。詳細な互換情報は Nugget の公式リポジトリを確認してください。

壁紙サイトと Nugget は同じですか？

別サービスです。壁紙ライブラリと Nugget 本体の両方をこのページから参照できます。
