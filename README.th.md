หน้านี้มีเป้าหมายเดียว: ใช้ **Nugget** เพื่อใช้งานวอลเปเปอร์เคลื่อนไหวบน **iPhone / iPad** โดยไม่ต้องเจลเบรก

## ลิงก์สำคัญ
- หน้าเว็บไซต์: [https://john-0x01.github.io/nugget-wallpaper/](https://john-0x01.github.io/nugget-wallpaper/)
- คลังวอลเปเปอร์: [https://mwallx.com](https://mwallx.com)
- ที่เก็บ Nugget: [https://github.com/leminlimez/Nugget](https://github.com/leminlimez/Nugget)

## Nugget คืออะไร?
Nugget เป็นเวิร์กโฟลว์เครื่องมือสำหรับปรับแต่งอุปกรณ์ Apple รวมถึงการนำเข้าวอลเปเปอร์เคลื่อนไหวไปยัง iPhone / iPad

หน้านี้อธิบายแนวทางแบบไม่เจลเบรก สำหรับผู้ใช้ที่ต้องการเอฟเฟกต์วอลเปเปอร์เคลื่อนไหวโดยไม่แก้ไขสิทธิ์ระบบระดับลึก

## คลังวอลเปเปอร์คืออะไร?
คลังวอลเปเปอร์อยู่ที่ [mwallx.com](https://mwallx.com)  
คุณสามารถดาวน์โหลดทรัพยากรวอลเปเปอร์เคลื่อนไหวเพื่อนำไปใช้ร่วมกับ Nugget ได้

## เริ่มต้นอย่างรวดเร็ว
1. เปิดคลังวอลเปเปอร์และดาวน์โหลดวอลเปเปอร์เคลื่อนไหวที่ต้องการ
2. เปิด Nugget และทำตามคำแนะนำใน repository เพื่อนำเข้าวอลเปเปอร์
3. เปิดใช้งานวอลเปเปอร์ในหน้าตั้งค่าของ iPhone / iPad เพื่อดูเอฟเฟกต์เคลื่อนไหว

## คำถามที่พบบ่อย
### 1. ต้องเจลเบรกไหม?
ไม่ต้อง คู่มือนี้เป็นแนวทาง Nugget แบบไม่เจลเบรก

### 2. รองรับอุปกรณ์อะไรบ้าง?
รองรับหลัก ๆ คือ iPhone และ iPad (iOS / iPadOS)  
โปรดตรวจสอบรายละเอียดความเข้ากันได้ล่าสุดจาก repository ของ Nugget

### 3. คลังวอลเปเปอร์กับ Nugget เป็นบริการเดียวกันหรือไม่?
ไม่ใช่ คลังวอลเปเปอร์ให้ทรัพยากร ส่วน Nugget ให้เครื่องมือและขั้นตอนการตั้งค่า

## หมายเหตุ
หน้านี้เป็นหน้าให้ข้อมูลและลิงก์ไปยัง Nugget และคลังวอลเปเปอร์ และไม่มีความเกี่ยวข้องกับ Apple Inc.
