## Nugget とは？
Nugget は Apple デバイスのカスタマイズ向けツールワークフローで、iPhone / iPad へのライブ壁紙導入に利用できます。

このページでは、低レベル権限を変更せずにライブ壁紙を使いたいユーザー向けの非脱獄手順を案内します。

## 壁紙ライブラリとは？
壁紙ライブラリは [mwallx.com](https://mwallx.com) で公開されています。  
Nugget と組み合わせて使えるライブ壁紙素材をダウンロードできます。

## クイックスタート
1. 壁紙ライブラリで好きなライブ壁紙をダウンロードします。
2. Nugget を開き、リポジトリの手順に従って壁紙をインポートします。
3. iPhone / iPad の壁紙設定で有効化し、動きのある表示を確認します。

## よくある質問
### 1. 脱獄は必要ですか？
不要です。このガイドは非脱獄の Nugget ワークフロー向けです。

### 2. 対応デバイスは？
主に iPhone と iPad（iOS / iPadOS）です。  
詳細な互換性は Nugget リポジトリの最新情報を確認してください。

### 3. 壁紙ライブラリは Nugget と同じですか？
違います。壁紙ライブラリは素材提供、Nugget はツールと導入手順を提供します。
